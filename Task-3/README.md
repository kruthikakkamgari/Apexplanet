# Task 3: Deep-Dive Analysis & Interactive Dashboard

## Objective
To perform a deep-dive analysis and build an interactive dashboard using Power BI.

## Key KPIs
- Total Revenue
- Average Purchase Value
- Customer Segmentation
- Revenue by Country

## Dashboard Features
- Interactive country slicer
- KPI cards
- Revenue and segmentation visuals

## Tools Used
- Python (Pandas, Scikit-learn)
- Power BI
- GitHub

## Outcome
An interactive dashboard that allows dynamic exploration of customer revenue and segments.
