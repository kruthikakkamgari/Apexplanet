# 📊 Data Storytelling & Statistical Validation

## 📌 Project Overview
This project focuses on analyzing business data to evaluate the impact of a new website design on customer conversion rates and revenue. Using data storytelling and hypothesis testing, we derive meaningful insights to support business decisions.

---

## 🎯 Objective
- Compare **Old vs New Website Performance**
- Analyze **Conversion Rates & Revenue**
- Validate results using **Statistical Testing (T-test)**

---

## 🛠️ Technologies Used
- Python 🐍
- Pandas
- Matplotlib
- SciPy

---

## 📂 Dataset Features
- customer_id
- month
- website_version (Old/New)
- visits
- conversions
- revenue

---

## ⚙️ Process
1. Data Loading & Preprocessing  
2. Feature Engineering (Conversion Rate)  
3. Data Analysis & Insights  
4. Hypothesis Testing (T-test)  
5. Visualization  
6. Result Export to File  

---

## 📊 Key Insights
- New website has **higher conversion rate**
- Revenue significantly increased with new version
- Customer engagement improved

---

## 📈 Hypothesis Testing
- **Null Hypothesis (H₀):** No difference in conversion rates  
- **Alternative Hypothesis (H₁):** New website improves conversion rates  

📌 Result:  
- P-value < 0.05 → **Statistically Significant**  
- New website performs better ✅  

---

## 📉 Visualizations
- Conversion Rate Comparison (Bar Chart)
- Revenue Comparison (Bar Chart)

---

## 📁 Output
- `business_insights.txt` → Stores analysis results

---

## 🚀 Conclusion
The new website design significantly improves both conversion rate and revenue. Businesses should consider fully adopting the new version.
