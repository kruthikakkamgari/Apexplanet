
# Employee Performance EDA Project

**Author:** Kruthika Akkamgari

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on an employee performance dataset.
It visualizes insights related to performance ratings, department trends, salary patterns, and correlations.

---

## 📂 Project Files Included
- employee_performance.csv  → Dataset (100 employee records)
- eda_analysis.py           → Python script for analysis
- eda_dashboard.png         → Combined dashboard image
- bar_chart.png             → Average Performance by Department
- pie_chart.png             → Performance Rating Distribution
- correlation_heatmap.png   → Correlation Matrix
- line_chart.png            → Average Salary vs Experience
- README.md                 → Project documentation

---

## 📊 Visualizations Included
1. Bar Chart – Average Performance Rating by Department  
2. Pie Chart – Performance Rating Distribution  
3. Correlation Heatmap – Relationships between numeric variables  
4. Line Chart – Average Salary vs Experience  

---

## ⚙️ Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

---

## ▶️ How to Run the Project

1. Install required libraries:
   pip install pandas numpy matplotlib seaborn

2. Run the Python script:
   python eda_analysis.py

---

## 🎯 Key Insights
- Marketing department shows slightly higher average performance.
- Performance ratings are fairly distributed across employees.
- Salary generally increases with experience.
- Weak correlations observed between performance rating and other numeric factors.
